# Which Meme Are You? 😂

A fun viral quiz website that tells you which internet meme matches your personality.

## 🌐 Live Website
https://Chin-si.github.io/which-meme-are-you/

## Features
- Meme personality quiz
- Animated results
- Meme images
- Download your meme result
- Mobile-friendly design
- Daily streak system
- Leaderboard
- Ad banner support for monetization

## Built With
- HTML
- CSS
- JavaScript

## Goal
Create a fun viral quiz that people can share and grow into a high-traffic site.
